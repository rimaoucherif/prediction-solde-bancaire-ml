# Prédiction de Solde Bancaire – Projet Machine Learning

## Description
Projet de Machine Learning visant à prédire le solde de clôture d’agences bancaires à partir de données historiques.

Le projet met en œuvre une pipeline complète de traitement des données : prétraitement, encodage, réduction de dimension (PCA), entraînement de modèles et évaluation des performances.

## Technologies utilisées
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

## Méthodologie
- Nettoyage et préparation des données
- Encodage des variables catégorielles
- Séparation Train / Test
- Standardisation
- Analyse en Composantes Principales (PCA)
- Comparaison de plusieurs modèles :
  - Régression Linéaire
  - Random Forest
  - Gradient Boosting
  - Decision Tree
  - KNN
  - AdaBoost

## Métriques d’évaluation
- R²
- RMSE
- MAE

## Résultats
Comparaison des performances des modèles avec et sans PCA afin d’identifier l’approche la plus pertinente.
